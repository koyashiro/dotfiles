## Installation

### Linux

```sh
git clone git@github.com:koyashiro/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

### Windows

```ps1
Set-ExecutionPolicy RemoteSigned
git clone git@github.com:koyashiro/dotfiles.git ~\.dotfiles
cd ~\.dotfiles
.\install.ps1
```
